### /deletebranch コマンドのプロンプト

いらないブランチをすべて削除する。以下の手順を実行する：

1. `git branch` でローカルブランチの一覧を確認する
2. `git branch -r` でリモートブランチの一覧を確認する
3. 現在のブランチ（`*` がついているブランチ）と `main` / `master` ブランチは削除しない
4. マージ済みのブランチや不要なブランチを特定する
5. ローカルブランチを削除する: `git branch -d <branch-name>` （マージ済みの場合）または `git branch -D <branch-name>` （強制削除の場合）
6. リモートブランチを削除する: `git push origin --delete <branch-name>` または `git push origin :<branch-name>`
7. 削除済みリモートブランチの参照をクリーンアップする: `git remote prune origin`

**注意事項**:

- 削除前に、ブランチに重要な変更がないことを確認する
- マージされていない変更があるブランチは、確認を取ってから削除する
- リモートブランチを削除する場合は、他の開発者に影響がないか確認する
